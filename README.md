# CalmaApp

CalmaApp is a calm, wellness-focused web app for daily check-ins, mood tracking, habit support, and guided breathing.

## Features

- Daily reflection prompts
- Mood tracking
- Habit checklists
- Simple progress summaries
- Breathing exercise view

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown in the terminal.

## Tech Stack

- React
- CSS
- Vite
